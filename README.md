Edelhaus Automotive 🏎️
A Premium Luxury Car Dealership & Management System

Edelhaus Automotive is a high-end web application designed for luxury car enthusiasts and dealership management. It combines a sophisticated, "Dark Luxury" user interface with robust functional features for browsing inventory, booking test drives, and managing dealership operations.

✨ Key Features
🛡️ Admin Dashboard
Inventory Management: Add, edit, or remove luxury vehicles from the collection.

Lead Tracking: View and manage user enquiries and service requests.

Analytics: Visual stats for top-viewed cars and booking trends.

Secure Access: Protected admin login system.

👤 User Experience
Dynamic Inventory: Browse cars with real-time filtering by brand (Ferrari, Lamborghini, Porsche, etc.).

Interactive Details: View high-resolution images, technical specs, and pricing in multiple currencies (INR/USD).

Booking System: Integrated forms for Test Drives and Service Appointments.

360° Interaction: (Optional/Planned) Interactive car views for an immersive experience.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom animations & Responsive Design)

Framework: Bootstrap 5 (Layout & Components)

Scripting: JavaScript (ES6+)

Data Handling: * JSON for static car data.

LocalStorage & IndexedDB for persistent user sessions and bookings.

Icons & Fonts: Bootstrap Icons, Google Fonts (Cinzel & Montserrat).
